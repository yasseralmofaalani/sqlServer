المحاضرة 2: تثبيت وإعداد SQL Server 2008

مقدم الدورة: م. ياسر المفعلاني

  • الصفحة الرئيسية
  • المحتوى

متطلبات النظام

المتطلبات الأساسية:

  • نظام التشغيل: Windows Server 2008 أو Windows Vista/7/8/10
  • المعالج: معالج 1.4 GHz أو أسرع (2.0 GHz موصى به)
  • الذاكرة: 512 MB على الأقل (2 GB موصى به)
  • مساحة القرص: 1.5 GB على الأقل للبرنامج الأساسي
  • .NET Framework: الإصدار 3.5 أو أحدث

خطوات التثبيت

الخطوة 1: تشغيل ملف التثبيت

  1. قم بتشغيل ملف setup.exe من قرص التثبيت
  2. اختر "New SQL Server stand-alone installation"
  3. سيتم فحص المتطلبات الأساسية

الخطوة 2: اختيار الميزات (Features)

اختر الميزات التي تريد تثبيتها:

  • Database Engine Services: المكون الأساسي (مطلوب)
  • SQL Server Replication: للنسخ المتماثل
  • Full-Text Search: للبحث النصي الكامل
  • Management Tools: أدوات الإدارة (موصى به بشدة)
// مثال على تثبيت الميزات الأساسية // تأكد من اختيار: // ✓ Database Engine Services // ✓ Management Tools - Basic // ✓ Management Tools - Complete

الخطوة 3: إعداد Instance

Instance هو نسخة من SQL Server. يمكن تثبيت عدة instances على نفس الخادم.

  • Default Instance: الاسم الافتراضي هو اسم الخادم
  • Named Instance: اسم مخصص مثل "SQLSERVER2008"

الخطوة 4: إعداد الخدمات (Services)

حدد الحسابات التي ستعمل عليها خدمات SQL Server:

  • SQL Server Database Engine: يمكن استخدام حساب النظام أو حساب مخصص
  • SQL Server Agent: لحساب المهام المجدولة

الخطوة 5: إعداد Authentication Mode

اختر طريقة المصادقة:

  • Windows Authentication Mode: استخدام حسابات Windows فقط
  • Mixed Mode: Windows Authentication + SQL Server Authentication
تحذير: إذا اخترت Mixed Mode، يجب تحديد كلمة مرور قوية لـ sa (system administrator).

الخطوة 6: إعداد Collation

Collation يحدد كيفية ترتيب ومقارنة النصوص. للعربية، استخدم:

  • Arabic_CI_AS: للعربية (Case Insensitive, Accent Sensitive)
  • SQL_Latin1_General_CP1_CI_AS: الافتراضي

إنشاء أول قاعدة بيانات

بعد التثبيت، يمكنك إنشاء قاعدة بيانات باستخدام SSMS:

الطريقة 1: استخدام الواجهة الرسومية

  1. افتح SQL Server Management Studio
  2. اتصل بالخادم
  3. انقر بزر الماوس الأيمن على "Databases"
  4. اختر "New Database"
  5. أدخل اسم قاعدة البيانات
  6. انقر "OK"

الطريقة 2: استخدام T-SQL

CREATE DATABASE قاعدة_البيانات_الأولى; -- مثال: CREATE DATABASE MyFirstDB;

الاتصال بـ SQL Server

للاتصال بـ SQL Server من SSMS:

  1. افتح SQL Server Management Studio
  2. في نافذة الاتصال، أدخل:
    • Server name: اسم الخادم أو IP (مثال: localhost أو .)
    • Authentication: Windows Authentication أو SQL Server Authentication
    • User name: إذا اخترت SQL Server Authentication
    • Password: كلمة المرور
  3. انقر "Connect"

التحقق من التثبيت

للتحقق من أن التثبيت تم بنجاح:

-- التحقق من إصدار SQL Server SELECT @@VERSION; -- عرض قواعد البيانات الموجودة SELECT name FROM sys.databases; -- عرض معلومات الخادم SELECT SERVERPROPERTY('MachineName') AS ServerName, SERVERPROPERTY('ProductVersion') AS Version, SERVERPROPERTY('ProductLevel') AS ProductLevel;
نصيحة: احفظ معلومات الاتصال في SSMS لتسهيل الوصول لاحقاً.

تثبيت SQL Server Management Studio

SSMS قد لا يكون مثبتاً مع SQL Server. قم بتثبيته منفصلاً:

  1. حمّل SQL Server Management Studio من موقع Microsoft
  2. شغّل ملف التثبيت
  3. اتبع خطوات التثبيت

الخلاصة

  • تأكد من تلبية متطلبات النظام قبل التثبيت
  • اختر الميزات المناسبة لاحتياجاتك
  • استخدم Mixed Mode إذا كنت بحاجة لـ SQL Server Authentication
  • اختر Collation مناسب للغة العربية
  • ثبت SQL Server Management Studio بشكل منفصل
