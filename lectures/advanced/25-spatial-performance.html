استخدام Spatial Indexes

الفهارس المكانية ضرورية لتحسين أداء استعلامات البيانات الجغرافية.

CREATE SPATIAL INDEX SIX_Location ON المواقع(Shape) USING GEOGRAPHY_GRID WITH ( GRIDS = (LEVEL_1 = MEDIUM, LEVEL_2 = MEDIUM, LEVEL_3 = MEDIUM, LEVEL_4 = MEDIUM), CELLS_PER_OBJECT = 16 );

تحسين استعلامات STDistance

-- استخدم Spatial Index مع STDistance DECLARE @point GEOGRAPHY = geography::Point(31.2357, 30.0444, 4326); DECLARE @radius FLOAT = 10000; SELECT * FROM المواقع WITH (INDEX(SIX_Location)) WHERE Shape.STDistance(@point) <= @radius;

الخلاصة

  • استخدم Spatial Indexes دائماً
  • استخدم WITH (INDEX) لتوجيه الاستعلام
  • راقب خطط التنفيذ
