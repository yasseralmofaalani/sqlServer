إنشاء Spatial Index

CREATE SPATIAL INDEX SIX_Location ON المواقع_الجغرافية(الموقع) USING GEOGRAPHY_GRID WITH ( GRIDS = (LEVEL_1 = MEDIUM, LEVEL_2 = MEDIUM, LEVEL_3 = MEDIUM, LEVEL_4 = MEDIUM), CELLS_PER_OBJECT = 16 );

تحسين الاستعلامات

الفهارس المكانية تحسن أداء استعلامات STDistance و STIntersects بشكل كبير.
