مقدمة عن QGIS

QGIS (Quantum GIS) هو نظام معلومات جغرافي مفتوح المصدر ومجاني. يمكن ربط QGIS مع SQL Server لاستخدام البيانات الجغرافية المخزنة في SQL Server.

متطلبات الربط

  • QGIS 3.0 أو أحدث
  • SQL Server 2008 أو أحدث
  • ODBC Driver for SQL Server
  • صلاحيات الوصول إلى قاعدة البيانات

إعداد ODBC Connection

في Windows:

  1. افتح "ODBC Data Source Administrator"
  2. اختر "System DSN" أو "User DSN"
  3. انقر "Add"
  4. اختر "SQL Server Native Client"
  5. أدخل اسم الاتصال ومعلومات الخادم
  6. اختر قاعدة البيانات
  7. اختبر الاتصال

إضافة طبقة من SQL Server في QGIS

الطريقة 1: استخدام Add Vector Layer

  1. افتح QGIS
  2. اذهب إلى "Layer" → "Add Layer" → "Add Vector Layer"
  3. اختر "Database"
  4. انقر "New" لإنشاء اتصال جديد
  5. اختر "MSSQL Spatial"
  6. أدخل معلومات الاتصال:
    • Service: اسم الخادم
    • Database: اسم قاعدة البيانات
    • Username: اسم المستخدم
    • Password: كلمة المرور
  7. انقر "OK"
  8. اختر الجدول المطلوب

الطريقة 2: استخدام Browser Panel

  1. في QGIS، افتح "Browser Panel"
  2. انقر بزر الماوس الأيمن على "MSSQL"
  3. اختر "New Connection"
  4. أدخل معلومات الاتصال
  5. اسحب الجدول إلى الخريطة

استيراد البيانات إلى SQL Server من QGIS

-- في QGIS، استخدم DB Manager: -- 1. Database → DB Manager -- 2. اختر الاتصال -- 3. Import Layer/File -- 4. اختر الملف أو الطبقة -- 5. حدد الجدول الهدف -- 6. انقر "OK"

تصدير البيانات من SQL Server إلى QGIS

يمكن تصدير البيانات مباشرة:

  1. افتح الطبقة من SQL Server في QGIS
  2. انقر بزر الماوس الأيمن على الطبقة
  3. اختر "Export" → "Save Features As"
  4. اختر الصيغة المطلوبة (Shapefile, GeoJSON, etc.)
  5. حدد الموقع واسم الملف
  6. انقر "OK"

استخدام SQL Queries في QGIS

يمكن تنفيذ استعلامات SQL مباشرة:

  1. افتح "DB Manager"
  2. اختر الاتصال
  3. اذهب إلى "SQL Window"
  4. اكتب الاستعلام
  5. انقر "Execute"
-- مثال استعلام في QGIS SELECT اسم_الموقع, Shape, عدد_السكان FROM المواقع WHERE عدد_السكان > 1000000;

إنشاء Views في QGIS

يمكن إنشاء Views من QGIS:

-- في DB Manager → SQL Window CREATE VIEW المدن_الكبيرة AS SELECT * FROM المواقع WHERE عدد_السكان > 1000000;

استخدام Plugins

هناك عدة Plugins مفيدة للربط مع SQL Server:

  • MSSQL Spatial: للربط المباشر مع SQL Server
  • DB Manager: لإدارة قواعد البيانات
  • QuickMapServices: لإضافة خرائط أساسية

أفضل الممارسات

  • استخدم ODBC Connection للربط
  • تأكد من أن البيانات الجغرافية في SRID 4326
  • استخدم Spatial Indexes لتحسين الأداء
  • احفظ الاتصالات للاستخدام المستقبلي
  • استخدم Views لتبسيط البيانات المعقدة
نصيحة: استخدم "Filter" في QGIS لتصفية البيانات قبل عرضها على الخريطة لتحسين الأداء.

استكشاف الأخطاء

  • تأكد من تثبيت ODBC Driver
  • تحقق من معلومات الاتصال
  • تأكد من صلاحيات المستخدم
  • تحقق من أن عمود البيانات الجغرافية موجود
  • راجع Log Messages في QGIS

الخلاصة

  • QGIS يمكنه الاتصال مع SQL Server عبر ODBC
  • يمكن استيراد وتصدير البيانات بسهولة
  • DB Manager يوفر واجهة لإدارة قاعدة البيانات
  • يمكن تنفيذ استعلامات SQL مباشرة
  • QGIS مجاني ومفتوح المصدر
