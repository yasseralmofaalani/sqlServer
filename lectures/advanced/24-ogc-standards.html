ما هو OGC؟

Open Geospatial Consortium (OGC) هو منظمة دولية تضع معايير للبيانات والخدمات الجغرافية. SQL Server 2022 يتبع معايير OGC.

WKT - Well-Known Text

-- تحويل إلى WKT SELECT Shape.STAsText() AS WKT FROM المواقع; -- إنشاء من WKT DECLARE @wkt NVARCHAR(MAX) = 'POINT(31.2357 30.0444)'; DECLARE @geo GEOGRAPHY = geography::STGeomFromText(@wkt, 4326);

WKB - Well-Known Binary

-- تحويل إلى WKB SELECT Shape.STAsBinary() AS WKB FROM المواقع;

الخلاصة

  • OGC يضع معايير للبيانات الجغرافية
  • WKT و WKB هما معياران أساسيان
  • SQL Server يتبع معايير OGC
