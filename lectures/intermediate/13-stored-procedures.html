المحاضرة 13: الإجراءات المخزنة (Stored Procedures)

مقدم الدورة: م. ياسر المفعلاني

  • الصفحة الرئيسية
  • المحتوى

إنشاء Stored Procedure

CREATE PROCEDURE GetStudents AS BEGIN SELECT * FROM الطلاب; END;

Stored Procedure مع معاملات

CREATE PROCEDURE GetStudentByID @StudentID INT AS BEGIN SELECT * FROM الطلاب WHERE رقم_الطالب = @StudentID; END;

تنفيذ Stored Procedure

EXEC GetStudents; EXEC GetStudentByID @StudentID = 1;
