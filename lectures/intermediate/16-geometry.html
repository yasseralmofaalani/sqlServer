المحاضرة 16: استخدام Geometry Data Type

مقدم الدورة: م. ياسر المفعلاني

  • الصفحة الرئيسية
  • المحتوى

إنشاء Point

DECLARE @point GEOMETRY; SET @point = geometry::STPointFromText('POINT(100 200)', 0); SELECT @point;

إنشاء LineString

DECLARE @line GEOMETRY; SET @line = geometry::STLineFromText( 'LINESTRING(0 0, 100 100, 200 200)', 0 );

إنشاء Polygon

DECLARE @polygon GEOMETRY; SET @polygon = geometry::STPolyFromText( 'POLYGON((0 0, 100 0, 100 100, 0 100, 0 0))', 0 );

الدوال الأساسية

-- STX و STY للحصول على الإحداثيات SELECT @point.STX AS X, @point.STY AS Y; -- STLength للحصول على الطول SELECT @line.STLength() AS الطول; -- STArea للحصول على المساحة SELECT @polygon.STArea() AS المساحة;

الخلاصة

  • Geometry للبيانات المسطحة
  • Point, LineString, Polygon هي الأنواع الأساسية
  • استخدم الدوال المدمجة للعمليات
