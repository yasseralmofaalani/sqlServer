ما هي البيانات الجغرافية؟

البيانات الجغرافية (Spatial Data) هي بيانات تمثل المواقع والأشكال الجغرافية على سطح الأرض.

Spatial Data Types في SQL Server 2022

SQL Server 2022 يدعم نوعين رئيسيين:

  • Geometry: للبيانات في المستوى الإقليدي (Flat/Euclidean)
  • Geography: للبيانات على سطح الكرة الأرضية (Ellipsoidal)

Geometry Data Type

يستخدم للبيانات في نظام إحداثيات مسطح مثل الخرائط.

CREATE TABLE المواقع ( رقم_الموقع INT PRIMARY KEY, اسم_الموقع NVARCHAR(100), الموقع GEOMETRY );

Geography Data Type

يستخدم للبيانات على سطح الكرة الأرضية مع نظام إحداثيات جغرافي.

CREATE TABLE المواقع_الجغرافية ( رقم_الموقع INT PRIMARY KEY, اسم_الموقع NVARCHAR(100), الموقع GEOGRAPHY );

أنواع البيانات المكانية

  • Point: نقطة واحدة (خط الطول، خط العرض)
  • LineString: خط يتكون من عدة نقاط
  • Polygon: مضلع مغلق
  • MultiPoint: عدة نقاط
  • MultiLineString: عدة خطوط
  • MultiPolygon: عدة مضلعات

إنشاء Point

-- إنشاء نقطة (Geometry) DECLARE @point GEOMETRY; SET @point = geometry::STPointFromText('POINT(100 200)', 0); -- إنشاء نقطة (Geography) - خط الطول، خط العرض DECLARE @geoPoint GEOGRAPHY; SET @geoPoint = geography::Point(31.2, 29.9, 4326); -- 31.2 خط الطول، 29.9 خط العرض، 4326 هو SRID (WGS84)

إدراج بيانات جغرافية

INSERT INTO المواقع_الجغرافية (رقم_الموقع, اسم_الموقع, الموقع) VALUES ( 1, 'القاهرة', geography::Point(31.2357, 30.0444, 4326) );

الخلاصة

  • Geometry للبيانات المسطحة
  • Geography للبيانات على سطح الكرة الأرضية
  • Point يمثل موقعاً واحداً
  • LineString يمثل خطاً
  • Polygon يمثل منطقة
